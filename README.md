# Transpiling Sopwith

"You can turn a cow into a hamburger. You cannot turn a hamburger back into the cow."

(play now!)</platform_web/www/index.html>

Sopwith was created as a demo for the BMB Compuscience Imaginet network, developed by David L. Clark.

https://en.wikipedia.org/wiki/Sopwith_(video_game)

Let's use Copilot to transpile the exe (not provided) to various targets.

For a playable version of 'modern' Sopwith, have a look at sdl-sopwith

https://github.com/fragglet/sdl-sopwith/blob/trunk/PHILOSOPHY.md

## Results

After multiple passes, a few screen shots of working and not working trials, this came out looking somewhat like Sopwith.

Missing some of the physics, collision details, and any semblance of cows.
